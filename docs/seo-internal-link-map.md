# HealthCalcHub SEO Internal Link Map

This document maps calculator hub pages to supporting blog posts and defines the core internal-link loops for the first content wave.

## Link architecture rules

- Every calculator page should link to 3 to 4 related calculators.
- Every calculator page should eventually surface 2 to 4 related blog posts near the FAQ or below the main content.
- Every blog post should link to 1 to 3 target calculators early in the article.
- Every blog post should link to 2 related articles in the same cluster.
- Cluster hub articles should link laterally to adjacent clusters where the search intent overlaps.
